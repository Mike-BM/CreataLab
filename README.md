# CreataLab

**About**

CreataLab is a modern web application built with React, Vite, and Tailwind CSS.

**Prerequisites:** 

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Create an `.env.local` file and set the environment variables (optional):

```
VITE_LOGO_URL=/logo.png
VITE_COMPANY_NAME=CreataLab
VITE_COMPANY_TAGLINE=Creative-Tech Innovation Lab
VITE_SOCIAL_INSTAGRAM=https://instagram.com/creatalab
VITE_SOCIAL_LINKEDIN=https://linkedin.com/company/creatalab
VITE_SOCIAL_FACEBOOK=https://facebook.com/creatalab
VITE_SOCIAL_TWITTER=https://twitter.com/creatalab
```

**Run the app:** `npm run dev`

**Build for production:** `npm run build`

**Preview production build:** `npm run preview`

**Tech Stack:**
- React 18
- Vite
- React Router
- Tailwind CSS
- Radix UI Components
- TanStack Query
- Framer Motion
